| Prefijo | px mínimo |
| ------- | --------- |
| `sm`    | `640px`   |
| `md`    | `768px`   |
| `lg`    | `1024px`  |
| `xl`    | `1280px`  |
| `2xl`   | `1536px`  |

Ejemplo:
<div class="bg-red-500 sm:bg-green-500 md:bg-blue-500 lg:bg-purple-500 xl:bg-orange-500 2xl:bg-emerald-500">
  Texto
</div>

<640px → bg-red-500
≥640px → bg-green-500
≥768px → bg-blue-500
≥1024px → bg-purple-500
≥1280px → bg-orange-500
≥1536px → bg-emerald-500
